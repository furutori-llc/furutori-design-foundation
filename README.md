# Furutori Design Foundation

島根県江津市の合同会社Furutoriが運営する、地方事業者向け汎用デザイン基盤。  
Web/LP・SNS投稿・提案資料・営業ツールまで横断的に対応する、案件可変型のデザインシステムです。

## このシステムの思想

地方事業者は飲食店・美容室・士業・小売・観光など業種の幅が広く、 
「ひとつのブランドトーン」では包摂しきれません。  
本基盤は4つのテーマを切り替える前提で設計された「土台」であり、  
案件ごとに最適なトーンへスイッチして使います。

## 4つのテーマ

| テーマID | 和名 | 向いているクライアント |
|---|---|---|
| `warm-local` | 地域温かみ系 | 飲食店・観光・地場産業・農家 |
| `btob-trust` | BtoB信頼系 | 士業・医療・建設・BtoB SaaS |
| `friendly-pop` | 親しみポップ系 | 美容・子育て・教育・小売 |
| `modern-refined` | 洗練モダン系 | カフェ・デザイナーズ・D2C |

各テーマの詳細は [`themes/`](./themes) を参照。

## 出力原則

- 日本語主体、和欧混植の美しさを優先
- モバイルファースト (375px基準)
- 配色は3色以内＋ニュートラル、アクセント1色
- AIっぽい典型デザイン (紫グラデ・過剰なガラスモーフィズム・意味のない3D) は禁止
- 詳細は [`guidelines.md`](./guidelines.md)

## ディレクトリ構成

```
.
├── tokens/         # 共通デザイントークン (色・タイポ・余白)
├── themes/         # 4テーマ別の設定
├── components/     # 最小限の共通コンポーネント
├── examples/       # 案件雛形 (LP / SNSバナー / 提案資料)
└── guidelines.md   # 禁則事項とディレクティブ
```
