# Examples

案件雛形の置き場所。実際の案件を走らせる中で、  
繰り返し使う構成をここに蓄積していく。

## 予定している雛形

- `lp-local-restaurant/` — 地方飲食店向けLP (warm-local)
- `lp-btob-service/` — BtoB SaaS/士業向けLP (btob-trust)
- `sns-banner-instagram/` — Instagramフィード用バナー各テーマ版
- `proposal-slide/` — 提案資料テンプレート

## 追加のタイミング

テンプレ化する価値があると判断した案件ベースのコードを  
そのクライアント情報を抜いた形でここに移植する。

単発の案件はここではなく個別Repoで管理する。
