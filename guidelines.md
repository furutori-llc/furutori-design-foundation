# Design Guidelines

Claude Designや他のAIツールでデザインを生成する際の共通ディレクティブ。  
案件ごとに `themes/` のいずれかと組み合わせて使う。

## 出力要件 (全テーマ共通)

### タイポグラフィ

- 日本語フォント前提 (Noto Sans JP / Noto Serif JP / Zen Kaku Gothic New / Shippori Mincho 等のwebfontを想定)
- 英字のみの見出しは避け、和欧混植の美しさを優先
- 見出しと本文でフォントファミリーを分ける (例: 見出しMincho系 × 本文Sans系)
- 行間は本文1.7〜1.8、見出し1.3〜1.4を基準

### レイアウト

- モバイル (375px) で破綻しないことが絶対条件
- 余白は広め、情報密度は中〜低
- センター揃えは見出しのみ。本文・箇条書きは左揃え基本
- 1画面1メッセージを意識。詰め込まない

### カラー

- 配色は3色以内＋ニュートラル、アクセント1色の原則
- グラデーションを使う場合は同系色の微妙なトーン差に留める (虹色・紫×ピンク等のAI臭は禁止)
- コントラスト比はWCAG AA準拠 (本文4.5:1以上)

### 装飾

- ドロップシャドウは使うなら1種類に統一 (深さ別の乱用禁止)
- 絵文字の多用禁止 (コーポレート用途では特に)
- 装飾的な斜めストライプ・放射状背景は文脈に合うときだけ

## 禁則事項 (AIっぽさ排除のため)

以下は明示的に避ける：

- 紫〜ピンクのグラデーション背景
- 意味のない3Dオブジェクト・浮遊する球体
- 過剰なガラスモーフィズム (backdrop-filter乱用)
- ストックフォト感のあるプレースホルダー画像 (笑顔のビジネスパーソンが握手している系)
- 「Lorem ipsum」的な無意味な英語ダミーテキスト (日本語ダミーを使う)
- テンプレ的なヒーローセクション (大きな英語見出し + 短い説明 + CTAボタン2個の定型)
- AIで生成しましたと一目でわかる等間隔グリッドカード配置
- Inter / Roboto / Arial などジェネリックフォント

## 実装時のチェックリスト

生成前に以下を確認：

- [ ] どのテーマ (`warm-local` / `btob-trust` / `friendly-pop` / `modern-refined`) を採用するか決まっているか
- [ ] クライアント業種・ターゲット顧客像が言語化されているか
- [ ] モバイル表示を前提とした構成になっているか
- [ ] 日本語コピーで設計されているか (英語は補助的にのみ使用)

## 案件別のトーン指定例

Claude Designのプロンプト内で以下のように指定する：

```
テーマ: warm-local
業種: 島根県益田市の蕎麦屋
ターゲット: 40-60代の地元客と観光客
必要な画面: LP (TOP / メニュー / 店舗情報 / アクセス)
トーン: 昔ながらの温かみ、派手さよりも信頼感
画像素材: 店構え1枚、料理写真3枚あり
```

このくらいの粒度で渡すと、基盤の原則＋テーマ設定＋案件個別指示が合わさって良質な出力になる。
